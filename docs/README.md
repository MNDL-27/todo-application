# TaskFlow - Demo Website

This is the landing page/demo website for the TaskFlow To-Do Application.

Visit the live demo at: [https://mndl-27.github.io/todo-application/](https://mndl-27.github.io/todo-application/)

## Local Development

To run locally:

```bash
cd docs
python3 -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
