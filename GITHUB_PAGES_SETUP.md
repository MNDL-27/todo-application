# üåê GitHub Pages Setup Guide

Follow these steps to enable GitHub Pages for your demo website:

## Step 1: Go to Repository Settings

1. Visit: https://github.com/MNDL-27/todo-application
2. Click on **Settings** (in the repository menu)

## Step 2: Navigate to Pages

1. In the left sidebar, scroll down and click on **Pages**

## Step 3: Configure Source

1. Under **Source**, select:
   - **Branch**: `main`
   - **Folder**: `/docs`
2. Click **Save**

## Step 4: Wait for Deployment

- GitHub will take 1-2 minutes to build and deploy your site
- You'll see a notification with your site URL
- Your demo will be available at: **https://mndl-27.github.io/todo-application/**

## Step 5: Verify

- Click on the provided URL or visit: https://mndl-27.github.io/todo-application/
- Your beautiful demo landing page should be live! üéâ

## Troubleshooting

If the site doesn't load:
- Wait a few more minutes (first deployment can take up to 10 minutes)
- Check the **Actions** tab to see if the deployment is still in progress
- Make sure the branch is set to `main` and folder is `/docs`
- Clear your browser cache and try again

## Future Updates

Whenever you make changes to files in the `docs/` folder:
1. Commit and push to GitHub
2. GitHub Pages will automatically redeploy within 1-2 minutes
